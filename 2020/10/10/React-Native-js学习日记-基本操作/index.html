定义组件

两种方式

const Example = () =>{
  return();
}

或者

function Example() {
    return ();
}

带上props就变成

const Example = (props) =>{
  return();
}

或者

function Example(props) {
    return ();
}

使用方法

编写好的组件可以用标签的形式进行复用
例如

const Demo = () =>{
  return(
<Text>demo</Text>
)}

即可通过<Demo></Demo>或者</Demo>进行使用

Props的作用

用来给自定义组件添加属性。使用方法为 ：

import React from 'react';
import { Text, View } from 'react-native';
function Cat(props) {
  return (
    <View>
      <Text>Hello, I am {props.name}!</Text>
    </View>
  );
}
export default function Cafe() {
  return (
    <View>
      <Cat name="Maru" />
      <Cat name="Jellylorum" />
      <Cat name="Spot" />
    </View>
  );
}

这里为Cat组件添加了name属性

需要注意的点：在定义函数时，xml代码内涉及到props的属性等代码时需要打{ }，例如定义时的props.name、状态传入了非串行值（数组、JS对象）等。
