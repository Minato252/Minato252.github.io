[React-Native.js学习日记]基本操作（props、state）

发表于 2020-10-10 分类于 学习记录

定义组件

两种方式

const Example = () =>{
  return();
}

或者

function Example() {
    return ();
}

带上props就变成

const Example = (props) =>{
  return();
}

或者

function Example(props) {
    return ();
}

使用方法

编写好的组件可以用标签的形式进行复用
例如

const Demo = () =>{
  return(
<Text>demo</Text>
)}

即可通过<Demo></Demo>或者</Demo>进行使用

Props的作用

用来给自定义组件添加属性。使用方法为 ：

import React from 'react';
import { Text, View } from 'react-native';
function Cat(props) {
  return (
    <View>
      <Text>Hello, I am {props.name}!</Text>
    </View>
  );
}
export default function Cafe() {
  return (
    <View>
      <Cat name="Maru" />
      <Cat name="Jellylorum" />
      <Cat name="Spot" />
    </View>
  );
}

这里为Cat组件添加了name属性

需要注意的点：在定义函数时，xml代码内涉及到props的属性等代码时需要打{ }，例如定义时的props.name、状态传入了非串行值（数组、JS对象）等。

[React-Native.js学习日记]环境搭建+运行

发表于 2020-10-10 更新于 2020-10-14 分类于 学习记录

环境搭建

我使用的工具有：

  • Node.js最新版本
  • jdk1.8（其他版本无效）
  • Genymotion（安卓虚拟机）
  • Android Studio
    搭建环境具体步骤参考

新建项目

在需要新建项目的文件夹内打开cmd，输入react-native init ProjectName

运行方法

  1. 打开Genymotion安卓虚拟机并联网(首次联网)
  2. 在项目文件夹内打开cmd，输入react-native start react-native run-android
  3. 虚拟机内将按照App.js中代码显示界面。更改其中代码，双击键盘RR可刷新。

(如果是github上下载的项目，需要先输入npm install再进行以上步骤）
